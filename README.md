<h1>PT_BR</h1>

<p>Esse é um Mod do Minecraft feito com intuito educacional, onde exploro e estudo programação em Java. </p>
<p>O intuito do mod é criar NPCS com interações, comidas, objetos, blocos e objetos utilizáveis para entender como cada classe de items funciona.</p>

<p>Esse projeto também será utilizado como Projeto de Extracurricularização em meu curso de Análise e Desenvolvimento de Sistemas na FATEC.</p>

<p>A musica dos Discos não está inclusa nos commits por conta de direitos autorais.</p>

<p>Você pode adicionar suas próprias músicas colocando o arquivo .ogg (obrigatoriamente em mono) na pasta sounds. Basta criar a pasta "sounds" no caminho  src/main/resources/assets/berinjela_punx/(crie a pasta sounds aqui) 
</p>
<p>IMPORTANTE: O arquivo .ogg deve ser nomeado "pawn_shop.ogg", caso contrário, não funcionará no respectivo disco.</p>


<h1>EN_US</h1>

<p>This is a Minecraft Mod made with the purpose of studying Java coding.</p>
<p>This mod will contain NPCS, foods, objects, blocks, Custom Items and more, so I can really explore how it is to code using Java.</p>
<p>This is also a project from my university, where I am currently studying System Analysis and Development (FATEC SP)</p>

<p>The music from the music discs is not included due to copyright. You can add your own music using .ogg files (mono audio only)</p>
<p>Just go to the path src/main/resources/assets/berinjela_punx/ and create the "sounds" folder. You then can add your music file inside it. IMPORTANT: the .ogg file must be named pawn_shop, otherwise it will not work.</p>
